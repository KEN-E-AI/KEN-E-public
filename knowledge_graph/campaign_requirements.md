# Knowledge Graph Design: BigBets & Campaigns

## 1\. Executive Summary

This document details the design for integrating campaigns into the knowledge graph.

### BigBet Node

**BigBet**
**Purpose**: Describes an initiative that the company is pursuing in support of a larger goal.

| name | type | description | example |
| :--- | :--- | :--- | :--- |
| node\_id | string | A unique identifier for the node. Generated by the system when the node is created. | `bet_c6051eee55b647ab81a80ffab37295e2` |
| label | string | The node type in neo4j. | `BigBet` |
| display\_name | string | A short name to describe the big bet in less than 60 characters. | `Launch New UX` |
| description | string | The full description of the big bet, including a summary of how it supports the goal. | `Simplify experience to reduce churn` |
| owner | string | (optional) The user name of the person who is responsible for the success of this campaign. | `Ken Williams` |
| created\_time | timestamp | The timestamp when the node was created. | 2025-07-29 22:06:45.928000 UTC |
| last\_modified | timestamp | The timestamp when the node was last modified | 2025-07-29 22:06:45.928000 UTC |
| created\_by | string | Identifies the user who approved the creation of the node, or identifies it as system generated. | `System generated` |
| last\_modified\_by | string | Identifies the user who last modified the node. | `John Doe` |
| embedding | list | Stores the vector embeddings used for search. Initially set to a null value when the node is created. | |

**Relationships**
| Label | Relationship | Label | Description |
| :--- | :--- | :--- | :--- |
| BigBet | `-[:BELONGS_TO]->` | Account | Each child node of the Account must have the BELONGS\_TO relationship. |
| BigBet | `<-[:IS_SUPPORTED_BY]-` | Goal | Each BigBet will roll up to one or more company goals. |
| BigBet | `-[:HAS_OBJECTIVE]->` | Objective | Describes one objective that the big bet must accomplish to support the larger company goal. |
| BigBet | `-[:MEASURES_EFFECTIVENESS_WITH]->` | MetricFilteredMetric | Identifies the metric or filtered metric that is used as a key performance indicator (KPI) to determine if the big bet is effective at accomplishing its goal. |
| BigBet | `-[:MEASURES_EFFICIENCY_WITH]->` | MetricFilteredMetric | Identifies the metric or filtered metric that is used as a key performance indicator (KPI) to determine if the big bet is efficient at accomplishing its goal. |
